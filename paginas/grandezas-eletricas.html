Tensão = Volts (V)

Tensão elétrica, também conhecida como diferença de potencial, é a diferença de potencial elétrico entre dois pontos. Sua unidade de medida é o VOLT homenagem ao físico italiano Alessandro Volta.
Em termos simples, a tensão Elétrica de forma comparativa seria a força necessária para movimentar os elétrons e criar assim uma corrente elétrica.


Cálculos

Existem 3 calculos para encontrar o valor da tensão elétrica com base em outras grandezas

1º Corrente x Resistencia

2º Potencia / Corrente

3º Raiz da Potencia x Raiz da Resistencia

Informe apenas 2 Valores


Corrente = Amperes (I)

A corrente elétrica é o fluxo ordenado de cargas elétricas, que se movem de forma orientada em um condutor elétrico.O nome "ampère" é em homenagem ao físico e matemático francês André-Marie Ampère (1775-1836). Ampère é considerado um dos principais fundadores da ciência do eletromagnetismo.

Cálculos

Abaixo temos 3 tipos de calculos para encontrar o valor da tensão elétrica com base em outras grandezas

1º Tensão x Resistencia

2º Potencia / Tensão

3º Raiz da Potencia x Resistencia

Informe apenas 2 Valores
